¶导入部分

¶系统开发流程与AI局限性

  1. 项目环境搭建，技术选型，框架整合
  2. 设计产品原型，例如库表、接口等
  3. 前后端编码，将设计转换为实际的软件功能
  4. 测试部署上线

AI能完成编码部分，但是环境搭建部分还是得人工

AI在设计阶段也可以起到辅助作用

在测试部署阶段，AI可以辅助生成测试用例，编写部署所需要的脚本

  • 若伊功能：快速构建 通用模块 代码生成（尤其是对于需要重复编写的增删改查部分）

  • 若伊：基于SpringBoot+Bootstrap的极速后台开发框架

¶基础应用部分

  • 若伊搭建
  • 入门案例
  • 功能详解
  • 项目结构
  • 源码阅读
  • 二次开发
  • 通义灵码AI工具

¶项目实战部分

  • 帝可得-项目搭建
  • 帝可得-前端页面
  • 帝可得-点位管理
  • 帝可得-人员管理
  • 帝可得-设备管理
  • AI优化代码
  • AI生成代码
  • AI代码调错

¶拔高原理部分

  • 若伊框架的核心原理和底层运行机制分析
  • Velocity代码生成器
  • RBAC权限控制
  • 异步任务管理器
  • 自定义日志注解
  • AI生成注释
  • AI辅助阅读

¶前置知识

  • 关系型数据库 MySQL
  • 非关系型数据库Redis
  • 后端
    • Springboot
    • MyBatis
  • 前端
    • Vue3

  • 工具
