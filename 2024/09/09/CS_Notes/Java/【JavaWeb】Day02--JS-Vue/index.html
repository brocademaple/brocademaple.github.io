¶将JS引入HTML的两种方式

¶1.3 JS基础语法规则

  • 3种输出语句：window.alert() | document.write() | console.log()
  • 变量
    • JS是弱类型语言→ 变量可以存放不同类型的值
    • 变量命名规则
    • var | let | const
    • js数据类型：number | string | boolean | null | undefined
      • 使用typeof函数可以返回数据类型
  • JS中的运算符 ： 算术运算符 | 赋值运算符 | 比较运算符 | 逻辑运算符 | 三元运算符
    • JS运算规则与Java的区别：\ == 和 ===
  • JS是弱数据类型的语言→形参和返回值，都不需要类型声明

¶1.4 函数

  • 两种定义函数的语法

    • function add(a, b){
    return a + b;
}

    • 调用函数多传参：不影响函数正常调用，只需函数名正确即可

¶1.5 JS对象

  • 参考：JavaScript 对象 (w3school.com.cn)

  • 第一类：基本对象 1.5.1

    • Array→定义数组用的 属性和方法

    • String→属性、方法

    • 自定义对象

    • JSON→应用在前后台交互的数据载体、代替xml格式、代码演示json对象

    • // //定义json
    var jsonstr = '{"name":"Tom", "age":18, "addr":["北京","上海","西安"]}';
    //alert(jsonstr.name);

    // //json字符串--js对象
    var obj = JSON.parse(jsonstr);
    //alert(obj.name);

    // //js对象--json字符串
    alert(JSON.stringify(obj));

  • 第二类：BOM对象，浏览器对象模型，和浏览器相关的几个对象 1.5.2

    • 通过操作BOM对象的相关属性或者函数来操作浏览器的部分功能
    • 提供对象：Window | Navigator | Screen | History | Location
    • Window对象
      • 其方法、属性可以省略window.
      • window对象提供了获取其他BOM对象的属性：history | location | Navigator | Screen
      • window常用函数：alert() | comfirm() | setInterval() | setTimeout()
    • Location对象及其属性

  • 第三类：DOM对象，JS将HTML文档中每一个标签都封装成一个对象

    • DOM Document Object Model 文档对象模型
    • 封装的对象：Document | Element | Attribute | Text | Comment
    • HTML文档→DOM树的转换
    • 学习DOM的作用：改变HTML元素的内容/样式 | 对HTML DOM事件作出反应 | 添加和删除HTML元素
    • 学习DOM的核心：如何获取DOM中的元素对象（Element对象，即标签） | 如何操作Element对象的属性（即标签的属性）
      • Element对象→通过Document对象获取 而Document对象→通过window对象获取document对象提供：获取Element元素对象的api 有四个 比较常用的有getElementById()
    • 操作Element对象的属性
      • 通过查询文档 JavaScript 和 HTML DOM 参考手册 (w3school.com.cn) 资料

  • 1.5.4 案例实操：复选选项亮灯

    • 需求1：点亮灯泡→换图→操作图片的src属性→获取img标签对象
    • 需求2：将所有的div标签的标签体内容后加上：very good 且为红色字体→先获取原有内容，再进行内容的追加
    • 需求3：选中所有的复选框→通过name属性获得所有的checkbox标签→遍历所有checkbox标签→设置为选中状态
    • 完整代码1.5.4.6

¶1.6 JS事件

  • HTML事件：发生在HTML元素上的事情→
    • 例如：按钮被点击onclick、鼠标移动到元素上、输入框失去焦点
  • 事件绑定
    • 方式1：通过html标签中的事件属性进行绑定 <input type="button" id="btn1" value="事件绑定1" onclick="on()">
    • 方式2：通过DOM中Element元素的事件属性进行绑定 <input type="button" id="btn2" value="事件绑定2"> document.getElementById('btn2').onclick = function(){alert("按钮2被点击了..."); }
  • 常用事件
    • onclick→鼠标单击事件 | onblur→元素失去焦点 | onfocus→元素获得焦点 | onload→某个页面或图像被完成加载 | onsubmit→当表单提交时触发该事件 | onmouseover→鼠标被移到某元素之上 | onmouseout→鼠标从某元素移开
  • 案例：按钮操控灯亮暗
    • 需求1： 点击 “点亮”按钮 点亮灯泡，点击“熄灭”按钮 熄灭灯泡→绑定单击事件on()和off()，进行图像变化→修改的src
    • 需求2： 输入框鼠标聚焦后，展示小写；鼠标离焦后，展示大写→给标签绑定onfocus和onblur事件→获取原本输入框的内容，即使用标签对象的value值
    • 需求3：“全选”选中所有复选框，“反选”使所有复选框取消→给“全选”和“多选”按钮绑定事件checkAll()和reverse()→定义checkAll()和reverse()两个函数→将选中的框改为true/false

¶2 Vue

¶快速入门

  • 相对于html+css+js更高效的前端开发方式

  • MVVM(Model-View-ViewModel)的前端开发思想

    • Model: 数据模型，特指前端中通过请求从后台获取的数据
      • 通过Ajax来发起请求从后台获取
    • View: 视图，用于展示数据的页面，可以理解成我们的html+css搭建的页面，但是没有数据
      • ElementUI框架来替代HTML+CSS来更加方便的搭建View
    • ViewModel: 数据绑定到视图，负责将数据（Model）通过JavaScript的DOM技术，将数据展示到视图（View）上
      • 侧重于ViewModel部分开发的vue前端框架，用来替代JavaScript的DOM操作

  • 框架：一个半成品软件，是一套可重用的、通用的、软件基础代码模型。基于框架进行开发，更加快捷、更加高效

  • Vue对象的创建

    • el: 用来指定哪儿些标签受 Vue 管理。 该属性取值 #app 中的 app 需要是受管理的标签的id属性值
    • data: 用来定义数据模型
    • methods: 用来定义函数

¶Vue指令

指令作用
v-bind / :为HTML标签绑定属性值，如设置 href , css样式等 2.3.1
v-model在表单元素上创建双向数据绑定 2.3.1
v-on / @为HTML标签绑定事件（必须是vue对象中声明的函数，在methods属性中定义的事件） 2.3.2
v-if条件性的渲染某元素，判定为true时渲染,否则不渲染 2.3.3
v-else
v-else-if
v-show根据条件展示某元素，区别在于切换的是display属性的值（不满足条件的代码依然存在，只是设置css样式来控制标签不去显示）
v-for列表渲染，遍历容器的元素或者对象的属性

¶生命周期

¶生命周期方法/钩子方法
状态阶段周期
beforeCreate创建前
created创建后
beforeMount挂载前
mounted挂载完成
beforeUpdate更新前
updated更新后
beforeDestroy销毁前
destroyed销毁后
